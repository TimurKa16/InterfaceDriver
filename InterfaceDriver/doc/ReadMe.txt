
Программа для управления платой интерфейса

main():
    Инициализация
    Чтение ДУСов
    Разбор посылок по COM-порту
    Управление арретиром
    Чтение датчика температуры
    

Прерывания

CpuTimerIsr():
    Инкрементирует счетчики
    1 секунда = 1000 тиков

SciRxIsr():
    Приём и обработка байта, принимаемому по COM-порту

adca1_isr():
	Обработка АЦП

Алгоритм перепрошивки:
	Прошивка принимается по SCI в буффер firmwareBuffer в отдельной области RAM
	По принятию целой прошивки, прыгает в Загрузчик (другой проект)
	В Загрузчике проверяется наличие новой прошивки в отдельной области RAM
	Если есть прошивка, то прошивает
	Если нет прошивки, то прыгает в основную прошивку
	
Примечание:

	GPIO51 - обогрев Renishaw, разрешение на включение до питания
	GPIO52 - цифровое питание драйверов
	GPIO53 - силовое питание верхнего драйвера
	GPIO54 - силовое питание нижнего драйвера
	GPIO55 - дополнительное питание нижнего двигателя

	На образцах с российскими двигателями на двигатели подается напряжение 48 В,
	поэтому важно не включать (GPIO54) одновременно с (GPIO51, GPIO55)
	Иначе АКИС будет "Не исправен"!!!
